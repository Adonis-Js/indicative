Ensures the value of the field is after the expected
date.
 
This method will import link:https://date-fns.org/docs/isAfter[isAfter] function of date-fns.
 
[source, js]
----
const rules = {
  confCall: `after:`${new Date()}
}
 
// or
const rules = {
  confCall: [
    rule('after', new Date())
  ]
}
----
